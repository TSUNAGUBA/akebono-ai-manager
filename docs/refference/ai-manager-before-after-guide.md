# AIマネージャーのご紹介
## ― 導入すると、毎日の仕事は何がどう変わるのか ―

---

## はじめに

このドキュメントは、「AIマネージャー」というシステムを導入すると、職場の毎日が具体的にどう変わるのかを、システムの専門知識がない方に向けてご説明するものです。

先にいちばん大事なことをお伝えします。

> **AIマネージャーは、あなたを監視したり評価したりする仕組みではありません。**
> **あなたの仕事から「考えるための材料集め」と「報告のための作業」をなくし、本来の仕事に集中できるようにする仕組みです。**

---

## こんな困りごとはありませんか?

多くの職場で、次のような声を耳にします。

**現場で働くメンバーの声**

- 目の前の作業に追われて、毎日があっという間に終わる。振り返る余裕なんてない
- 自分の作業が、会社全体やお客様にとってどんな意味を持つのか、正直よくわかっていない
- お客様の業界の専門用語や商習慣がわからず、都度誰かに聞くか、わからないまま進めてしまう
- 日報や報告書を書く時間が、毎日の最後に重くのしかかる
- 「自分なりの仮説を持て」「考えて動け」と言われるが、何をどう考えればいいのかわからない
- 会社からAIツールを与えられたが、何に使えばいいのかわからず、結局開いていない

**経営者・リーダーの声**

- 仕事の依頼や相談が特定の人に集中し、その人がボトルネックになっている
- メンバー一人ひとりに声をかけて状況を聞き、次の動きを示す時間が取れなくなってきた
- 自分の頭の中にある「お客様との経緯」や「判断の基準」を、うまくメンバーに引き継げない
- メンバーが今どこで困っているのか、問題が大きくなるまで気づけない
- 人を増やす余裕はないのに、案件は増えていく

AIマネージャーは、これらの困りごとに正面から応えるために設計されたシステムです。

---

## AIマネージャーとは?

ひとことで言えば、**普段お使いのチャット(Google Chat)の中にいる「もう一人の頼れる先輩」**です。

新しいアプリを覚える必要はありません。いつものチャットで話しかけるだけで、次のことをしてくれます。

1. **教えてくれる** … お客様の業界知識、専門用語、過去の経緯を、聞けばすぐに答えてくれます
2. **段取りしてくれる** … 仕事の依頼を整理し、やることを分解して、順番を提案してくれます
3. **一緒に考えてくれる** … 朝と夕方に短い問いかけをして、あなたの考えを整理する手伝いをします
4. **書いてくれる** … 日報や報告書は、その日のやり取りから自動で作られます。あなたは確認するだけです

そして、もうひとつ大切な原則があります。

> **AIは「提案して、記録する」まで。「決めて、責任を持つ」のは、いつも人間です。**

AIがあなたを評価したり、順位をつけたり、勝手に仕事の指示を変えたりすることは、仕組みの上でできないように作られています。

## 導入前と導入後:1日はこう変わります

### 現場メンバーの1日

| 場面 | 導入前 | 導入後 |
|---|---|---|
| **朝・仕事の始まり** | とりあえず昨日の続きから着手。この作業が全体のどこにつながるかは、あまり意識できていない | チャットにAIから短いメッセージが届く。「今日のこの作業は、〇〇社の△△プロジェクトの一部です。終わったとき、どうなっていれば成功だと思いますか?」——答えに迷えば、AIが背景を説明しながら一緒に考えてくれる |
| **作業中・わからないことが出た** | 詳しい人の手が空くのを待つか、わからないまま自己流で進める | その場でAIに質問。「種まきリストって何?」→「複数の注文の商品をまとめて集めてから、注文ごとに配り分けるためのリストです。給食の配膳をイメージしてください」——自分の日常に置き換えた説明で、すっと腹に落ちる |
| **仕事の受け取り** | 口頭やバラバラの連絡で依頼が来る。優先順位は自分で推測。抜け漏れが起きやすい | 依頼はAI経由で整理されて届く。やることの分解、期限の目安、関連する背景情報がセットになっている |
| **夕方・仕事の終わり** | 記憶を掘り起こしながら日報を書く。20〜30分かかる日も。書くこと自体が目的化しがち | AIが「朝の予想と比べて、実際はどうでしたか?」と一言聞いてくる。答えると、その日のやり取りから日報が自動で完成。**確認ボタンを押すだけ** |
| **数か月後** | 忙しかった記憶はあるが、何が身についたかは曖昧 | 「自分の予想と実際の結果」の記録が積み上がっている。自分の考えがどう当たり、どう外れたかを振り返れる、**自分だけの成長の記録**になっている |

### 経営者・リーダーの1日

| 場面 | 導入前 | 導入後 |
|---|---|---|
| **メンバーの状況把握** | 一人ひとりに声をかけてヒアリング。時間も気力も使うわりに、全体像はつかみきれない | 週次でAIからサマリが届く。「全案件の進捗、停滞しているポイント、判断が必要な事項」が一覧に。声かけは、本当に必要な相手に絞れる |
| **知識・判断の引き継ぎ** | 頭の中にしかない。「あの人に聞かないとわからない」状態が続く | 思いついたときにチャットで「これ、ナレッジに入れておいて」と話すだけ。AIが整理して蓄積し、以後は**メンバー全員がいつでもその知識にアクセスできる** |
| **迷いどころの判断** | 問題が大きくなってから報告が上がってくる | AIが「これは判断が必要です」と早めにエスカレーション。判断した結果はAIが記憶し、**次から同じ場面ではAIが同じ基準で案内してくれる**——判断するたびに、会社の資産が増えていく |
| **人が増やせない中での案件増** | 特定の人の残業と気合いで吸収 | 知識の供給・段取り・報告作成をAIが肩代わりすることで、**人を増やさずにチーム全体の処理量が上がる** |

## 何が変わって、何が変わらないのか

### 変わること

- **「調べる・思い出す・書き写す」時間が消えます。** 知識はAIが即答し、報告書は自動で下書きされます
- **仕事の全体像が、いつでも見えるようになります。** 「この作業は何のためか」を、AIが毎回教えてくれます
- **考える習慣が、無理なく身につきます。** 朝夕の短い問いかけに答えるうちに、「予想→実行→振り返り」のサイクルが自然と回り始めます
- **知識と判断基準が、会社の資産として貯まります。** 特定の誰かが不在でも、仕事が止まりにくくなります

### 変わらないこと

- **決めるのは人間です。** 優先順位の決定、お客様への約束、仕事の采配は、これまで通り人が行います
- **評価するのも人間です。** AIが人を採点したり、順位づけしたりすることはありません
- **あなたの働き方の主導権は、あなたにあります。** AIの提案は「採用する/しない」を選べます。むしろ「なぜ採用しなかったか」というあなたの判断こそが、チームの貴重な知恵として扱われます

## よくあるご質問

**Q. 結局、AIに監視されるのでは?**

A. いいえ。このシステムの記録は「誰かを評価するため」ではなく、「本人の振り返りの材料」と「リーダーが適切なタイミングで手を差し伸べるため」に使われます。この用途は社内ルールとして明文化することを推奨しており、システム上も、AIが人の優劣を出力することは禁止事項として作り込まれています。

**Q. AIに仕事を奪われませんか?**

A. AIが引き受けるのは「調べる」「段取りの下書き」「報告書の作成」といった、これまであなたの時間を奪っていた作業です。空いた時間は、判断・工夫・お客様との対話といった、人にしかできない仕事に使えるようになります。

**Q. 毎日の問いかけが、面倒な義務になりませんか?**

A. 問いかけは朝夕それぞれ数分で終わる設計です。しかも、答えることで日報が自動で完成するため、**トータルの手間はむしろ減ります**。うまく答えられない日があっても構いません。AIが背景を補いながら一緒に考える設計になっており、「答えられないこと」を責める仕組みではありません。

**Q. AIの言うことが間違っていたら?**

A. AIは確信が持てないとき「わかりません」と言い、責任者に判断を仰ぐように作られています。また、AIの提案が外れた場合は、その記録をもとに判断基準を修正していく運用を組み込んでいます。使うほどに、その会社に合ったAIに育っていきます。

**Q. 新しいツールの操作を覚えるのが苦手です。**

A. 操作は「いつものチャットに返事をする」だけです。専用画面の操作研修などは基本的に不要です。

## 導入の進め方(3ステップ)

一度にすべてを変えるのではなく、小さく始めて確かめながら広げます。

1. **ステップ1(約1か月): まず「対話と日報」だけ**
   朝夕の問いかけと日報の自動作成からスタート。まずは「報告の手間が減った」「聞けばすぐ教えてくれる」という体感を届けます
2. **ステップ2: 仕事の受け渡しをAI経由に**
   依頼の整理・段取り・進捗の共有をAIに集約。「誰が何をしているか」が自然に見える状態を作ります
3. **ステップ3: 見える化ダッシュボード**
   チーム全体の状況や、一人ひとりの振り返りの記録をグラフで確認できるようにします

## おわりに

AIマネージャーが目指すのは、AIが人の代わりに働く職場ではありません。

**「知っている人しか動けない仕事」を「誰もが考えて動ける仕事」に変え、人がもっと人らしい仕事——考えること、決めること、お客様と向き合うこと——に時間を使える職場**です。

チームの知識と判断は、日々の何気ないやり取りの中から自動的に積み上がり、会社の財産になっていきます。導入したその日から少しずつ、そして使い続けるほどに、その効果は大きくなります。
